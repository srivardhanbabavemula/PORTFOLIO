# Srivardhan Baba Vemula — Cinematic Portfolio

A cinematic Next.js portfolio for **Srivardhan Baba Vemula** — Data Engineer, Python, AWS, PySpark, Kafka, and ML systems.

## Run locally

From this folder (`cinematic-portfolio-main`):

```bash
npm install
npm run dev
```

Or from the parent folder (`cinematic-portfolio`):

```bash
npm install
npm run dev
```

Open **http://localhost:3000**

## Build

```bash
npm run build
npm start
```

## Main content files

- `data/profile.json` — profile, experience, GitHub projects, certifications, skills
- `data/content.json` — hero copy, section labels, footer text
- `public/assets/` — photos, intro/footer videos, resume & certificate PDFs
- `scripts/generate-intro-video.py` — regenerate cinematic intro videos from photos

## Deploy

Push to [GitHub PORTFOLIO](https://github.com/srivardhanbabavemula/PORTFOLIO) and import on [Vercel](https://vercel.com).
